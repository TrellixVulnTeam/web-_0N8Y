### 不可不知的解构赋值

#### 简介：深入原理讲解结构赋值及常见用法

- ##### 介绍

  解构赋值可以理解为赋值操作的语法糖，它是针对数组或者对象进行模式匹配，然后对其中变量进行赋值。代码书上言简意赅，语义明确，也方便了对象数据的读取操作。

- ##### 实质

  - ES6中只要某种数据有lterator接口(也就是可以循环迭代),都可以进行数组的解构赋值。

- ##### 使用场景

  - 数组结构

    ```js
    {
    let a,b,c
    [a,b,c]=[1,2]
    console.log(a,b,c) //1,2,undefined
    }
    {
        let a,b,c
        [a,b,c=6]=[1,2]
        console.log(a,b,c)//1,2,6
    }
    {
        let a,other
        [a,...other]=[1,2,3]
        console.log(a,other)//1,[2,3]
    }
    ```
