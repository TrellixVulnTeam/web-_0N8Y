### CSS的重要特性之重叠

#### 简介：介绍css中的选择器及层叠特性

- css全称：层叠样式表（Cascading Sytle Shees)
- 层叠是一个基本特征
  - 一个css属性被多次声明的时候，会根据优先级或声明顺序来计算采用那个样式
- 优先级是怎么计算的
  - id>class>tag
  - 标签选择器1
  - 类选择器10
  - id选择器100
  - 通配符选择器和表示关系的选择器，对优先级没有影响
- 列外规则
  - ！important
  - 注意的
    - 不利于代码维护
    - 建议不要在自己的局部代码（公用代码）里使用important
    - 可以在引入外部css，覆盖他本身的代码时，就可以用
